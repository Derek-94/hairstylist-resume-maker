## 이런 분들을 위해 만들었어요

- 이력서를 한 번도 써본 적 없는 미용학원 졸업생
- 메모장에 적은 자기소개를 사진 찍어 문자로 보내던 분
- 어떤 양식을 써야 할지 몰랐던 20대 초반 취준생

---

## 주요 기능

- **설문 형식 입력** — 한 화면에 질문 하나씩, 막히는 부분 없이 끝까지
- **포트폴리오 사진 첨부** — 내가 한 머리 스타일을 사진으로 보여주세요
- **4가지 디자인 템플릿** — 내 데이터가 들어간 상태로 골라보세요
- **PDF 저장** — 버튼 하나로 이력서 PDF가 내 폰/PC에 저장

---

## 서비스 흐름

```
이름 입력 → 생년월일 → 성별 → 연락처
→ 프로필 사진 → 보유 기술 선택 → 포트폴리오 사진
→ 경력 · 자격증 · 자기소개
→ 미리보기 (디자인 선택) → PDF 저장
```

## 템플릿 미리보기

| 미니멀 | 다크 | 웜 | 볼드 |
|--------|------|----|------|

## 시작하기

```bash
# 저장소 클론
git clone https://github.com/Derek-94/hairstylist-resume-maker.git
cd hairstylist-resume-maker

# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
```

브라우저에서 `http://localhost:5173` 접속
