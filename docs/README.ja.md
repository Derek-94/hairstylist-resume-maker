## こんな方のために作りました

- 履歴書を一度も書いたことがない美容学校の卒業生
- 自己紹介を手書きして写真を撮り、採用担当者にテキストで送っていた方
- どのフォーマットを使えばいいかわからない20代前半の求職者

---

## 主な機能

- **アンケート形式の入力** — 1画面に1つの質問、つまずかずに最後まで
- **ポートフォリオ写真の添付** — 自分が手がけたヘアスタイルの写真を見せましょう
- **4種類のデザインテンプレート** — 自分のデータが入った状態でデザインを選択
- **PDF保存** — ボタン一つで履歴書PDFがスマホ・PCに保存

---

## 利用の流れ

```
名前 → 生年月日 → 性別 → 電話番号
→ プロフィール写真 → 保有スキル選択 → ポートフォリオ写真
→ 職歴・資格・自己紹介
→ プレビュー（デザイン選択）→ PDF保存
```

## はじめ方

```bash
git clone https://github.com/Derek-94/hairstylist-resume-maker.git
cd hairstylist-resume-maker

pnpm install
pnpm dev
```

ブラウザで `http://localhost:5173` を開いてください。
